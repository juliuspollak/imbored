# v73

- Hidden users can no longer create teams, enforced in both UI and database RLS.
- Leaving a team now uses a server-side function; when the final member leaves, the empty team is deleted automatically.
- Replaced the plain Leave link with a modern pill action and confirmation.
- Added team emojis with automatic name-based suggestions and an emoji picker.
- Submitted wishes can now be edited until an admin prices, approves or declines them.
- Added a modern inline wish editor and richer wish status presentation.
